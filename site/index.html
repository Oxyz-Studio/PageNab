FAQ

Frequently asked.

Does PageNab send my data anywhere?

No. PageNab is 100% local. Captured data goes to your clipboard and your Downloads folder. Nothing is sent to any server. There is no analytics, no telemetry, no tracking.

What about sensitive data like passwords and tokens?

PageNab automatically sanitizes sensitive data before capture. Authorization headers, API keys, and tokens are stripped from network requests. Password field values are replaced with ***. Cookie values with sensitive names (token, session, auth, key, secret, password) are masked. Input field values in interactions are always masked.

What's the difference between the three presets?

Light (default): screenshot + metadata + console errors/warnings + failed network requests (~200–500 tokens). Full: everything including DOM, cookies, storage, interactions, performance (~1.5K–150K tokens). Custom: you choose exactly which data types to include.

Which AI assistants work with PageNab?

Any assistant that accepts text and images from the clipboard. This includes Claude Code (CLI), Claude.ai, ChatGPT, Cursor, Windsurf, GitHub Copilot, and more. When you paste, both the text data and the screenshot image are attached.
