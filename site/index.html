FAQ

Frequently asked.

01How do I capture console errors and network requests to share with an AI assistant?

Click the PageNab icon or press the keyboard shortcut. It captures a screenshot, console logs (errors, warnings, stack traces), failed and slow network requests, and page metadata — all in one click. Everything is copied to your clipboard as structured text + image. Just paste into any AI assistant. No need to open DevTools, no MCP server, no setup.

02How is PageNab different from Chrome DevTools MCP?

Chrome DevTools MCP requires a running MCP server, JSON configuration, and an AI tool that supports the MCP protocol. PageNab is a standard Chrome extension: install it, click, paste. It works with any AI that accepts text and images from your clipboard — web apps, CLI tools, IDE extensions. Zero configuration, zero dependencies.

03Can I use PageNab to debug a web app with ChatGPT, Claude, or Cursor?

Yes. Capture your page, then paste — your AI gets the screenshot plus structured context: console errors, failed network requests, page metadata, and more. It works with ChatGPT, Claude, Gemini, Cursor, GitHub Copilot, Windsurf, Perplexity, and any tool that accepts text and images from the clipboard.

04What sensitive data is stripped before capture?

Authorization headers, API keys, and tokens are automatically removed from network requests. Password fields are replaced with ***. Cookies with sensitive names (token, session, auth, secret) are masked. User input values in interactions are always hidden. Everything stays on your machine — nothing is sent to any server.

05Can I capture a specific element or area instead of the full page?

Yes. PageNab offers three capture modes: Full Page, Area (draw a rectangle to select a region), and Element (click to select a specific DOM element). The cropped screenshot and dev tools context are copied together. The full-page screenshot is also saved to your Downloads folder.

06Is PageNab free?

Yes, 100% free and open source under the MIT license. No account required, no usage limits, no premium tier. The source code is on GitHub — you can audit every line.
